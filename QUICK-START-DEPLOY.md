Quick start — Vercel deployment

This template deploys as a static site on Vercel.

If this template used server-side mail scripts, contact-form migration is required (mail.php/sendemail.php). See session notes.

Required environment variables (if any):
- SMTP_HOST
- SMTP_USER
- SMTP_PASS

Deployment steps:
1. Create a new Vercel project and connect to this repo/folder.
2. Add any environment variables in Vercel settings.
3. Deploy and verify previews.
